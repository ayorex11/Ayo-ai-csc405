Quick Reference

Cheat Sheet

Key formulas and facts — read this the morning of your exam.

🗺️ Search

BFS = Queue (FIFO). Shortest path. High memory.

DFS = Stack (LIFO). Low memory. Not shortest.

A* score:

f(n) = g(n) + h(n)

Manhattan distance:

h = |r1-r2| + |c1-c2|

♟️ Game Playing

Minimax: MAX picks highest, MIN picks lowest. Back-propagate from leaves.

Alpha-Beta: Same result as Minimax but skips useless branches.

α = best MAX found so far (↑ only)

β = best MIN found so far (↓ only)

📜 Logic

∀x = "for all x"

∃x = "there exists x"

⇒ = "implies (if...then)"

Modus Ponens:

P is true. P⇒Q is true. ∴ Q is true.

🌐 Bayes Nets

Arrow A→B means A influences B.

Each node has a CPT showing its probability given its parents.

Conditional independence: knowing C makes A & B irrelevant to each other.

🤖 RL / MDPs

Q-Learning update:

Q(s,a) += α[R + γ·maxQ' − Q(s,a)]

Bellman equation:

v*(s) = max_a[R + δ·Σ P·v*(s')]

🎓 ML Metrics

Precision:

TP / (TP + FP)

Recall:

TP / (TP + FN)

In medicine: Recall matters more (don't miss sick patients!)
